# Fitness Tracker

A professional full-stack web application designed to help users track their daily workout routines. 

## Features
- **Secure Authentication**: User login and registration powered by JWT.
- **Workout Management**: Create, view, and delete personal workout entries.
- **Clean Interface**: Modern UI built with a mobile-first approach.
- **RESTful API**: Seamless communication between the frontend and backend.

## Tech Stack
- **Frontend**: HTML5, CSS3 (Modern Flexbox), JavaScript (Async/Await)
- **Backend**: Node.js/Express
- **Database**: MongoDB

## Deployment
This project is deployed and publicly accessible for industry practice evaluation.
